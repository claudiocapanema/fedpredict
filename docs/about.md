# About

FedPredict is an open-source project freely available on [Github](https://github.com/claudiocapanema/fedpredict) under the BSD-3 license.
The project is led by Cláudio G. S. Capanema and has been developed across the laboratories WISEMAP (UFMG), H.IAAC (UNICAMP), and NESPED-Lab (UFV).
